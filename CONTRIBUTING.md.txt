## How to Contribute
1. **Fork the Repository**: Click the "Fork" button on GitHub to create your own copy.
2. **Clone to Your Machine**: Use GitHub Desktop to clone your fork.
3. **Create a Branch**: Make changes in a new branch (e.g., `feature/new-strategy`).
4. **Commit Changes**: Use GitHub Desktop to commit your work.
5. **Push and Submit a Pull Request**: Push your branch and create a pull request on the original repository.
## Guidelines
- Ensure your code follows MQL5 best practices.
- Test changes in MetaTrader 5 before submitting.
- Include a clear description in your pull request.
## Reporting Issues
- Use the [Issues](https://github.com/punlie/PunlieGridEA_v3/issues) tab to report bugs or suggest features.
## Contact
For questions, reach out via GitHub Issues or [punlie](https://github.com/punlie).
